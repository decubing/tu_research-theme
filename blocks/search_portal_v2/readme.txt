=== Skilltype Use Cases ===

Block for displaying a filtered list of use cases.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/skilltype-use-cases` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

== Development ==

1. `npm install`
2. `npm start` for development.
